# BorrowChecker Test Data
Sample data for https://github.com/araex/borrow-checker

## Domain Objects
### Entity
An individual or organization that pays expenses to be shared.

### Group
A group of entities that want to share expenses. 1 group owns N ledgers. 1 group is represented as 1 git repository. Group config is found in `./group.toml`

### Ledger
A list of expenses. Ledgers are represented as folders in `./ledgers`. Config per ledger is found in `./ledgers/{ledger_name}`.

### Expense
A single expense paid by a group member. The file name is a UUIDv7, generated by the client at time the expense was added, with .toml extension.
